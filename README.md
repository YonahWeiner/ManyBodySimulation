# ManyBodySimulation
## About
In highschool I learned about about Newtonian gravity and was excited to simulate it. I forgot that I wrote this, then 13 years later I rediscovered it. It works!
## Running the Program
In order to run the program, you need java installed on your computer. To run, navigate in the command line to the file GuiRun.java. Then write
<code>java GuiRun</code>
